## Syntax Highlighting

Automatically syntax highlighting:

````markdown
```js title="index.mdx"
console.log("hello, world");
```
````

Renders:

```js title="index.mdx"
console.log("hello, world");
```

You can also add the `showLineNumbers`, `title="index.js"` or `{line number, line number-line number}` modifier to highlight specific lines:

````markdown filename="index.mdx"
```jsx showLineNumbers title="index.js" {4, 7-8}
import useSWR from "swr";

function Profile() {
	const { data, error } = useSWR("/api/user", fetcher);

	if (error) return <div>failed to load</div>;
	if (!data) return <div>loading...</div>;
	return <div>hello {data.name}!</div>;
}
```
````
